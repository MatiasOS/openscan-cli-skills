# openscan-cli-skills

## Web Explorer

Browse blockchain data at [openscan.eth.link](https://openscan.eth.link):

- Network stats: `https://openscan.eth.link/#/:chainId` (e.g., `https://openscan.eth.link/#/1` for Ethereum)
- Transaction: `https://openscan.eth.link/#/:chainId/tx/:txHash`

## Installation

```sh
bun install
```
